# Simple eCommerce Service Sample

## 준비사항
- 샘플 데이터베이스(MySQL 8.x)
- 자바 개발 환경 11 이상

## 테크 구성
- Spring Boot 2.7
- Java 11
- MySQL 8.x
- JPA

## 주요 기능들
- Home
- Search
- Product List
- Product Detail
- Cart
- Checkout
- Thanks

## 프로젝트 기본 구조
- controller
- service
- repository

## Run Configuration
- Server Port: 8080
- Profile: local
